Tentang Saya

Perkenalkan nama saya Ratu Mutiara Kaila, saya biasa nya dipanggil Kaila or Ratu. Saya anak kedua dari dua bersaudara, saya seorang siswi SMAN 3 Kota Serang. Saya suka menggambar tapi saya tidak pandai menggambar, dan yang paling penting saya suka Rizwan Fadilah anak dari Bapak Sule. Saya bertekad dan berpegang teguh untuk menjadi kekasih Rizwan dan itulah impian saya, Aamiin gitu pak ehehe. Hobi saya yaitu jalan-jalan dan scrol tiktok.
